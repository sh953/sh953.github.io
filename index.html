Introduction

In recent years, police brutality has risen to become a major topic in political discourse, and today, we are looking at data collected by the Washington Post of each and every fatal police shooting from 2015 to just yesterday, December 15, 2017. First, we will be pulling the data from a csv file, then we will be processing the data, doing some exploratory analysis and creating data visualizations, and finally, we will be using machine learning to do some hypothesis testing.

Here we can use SciKit to perform Cross Validation, where we split our test data up into a section for testing and one for validating our testing. We can perform this task using the function cross_validation.KFold. If you would like to learn more about SciKit or KFold, see <a href= http://scikit-learn.org/stable/modules/generated/sklearn.model_selection.KFold.html> here </a>

In [418]:
# Code based off tutorial found at https://jonsm.github.io/
x_train = shootings_by_month['month']
y_train = shootings_by_month['count']
cv = cross_validation.KFold(len(x_train), n_folds=4)

validation_errors = []
train_errors = []
for degree in range(1,10):
    avg_train_error = 0
    avg_validation_error = 0
    for testcv, traincv in cv:
        #train our data with training subset
        cv_fit = np.polyfit(x_train[traincv], y_train[traincv], degree)
        cv_poly = np.poly1d(cv_fit)

        #test accuracy of data using squared error
        #training error
        predicted_train = cv_poly(x_train[traincv])
        train_error = (predicted_train - y_train[traincv])**2
        avg_train_error += np.mean(train_error)/4

        #validation error
        predicted_validation = cv_poly(x_train[testcv])
        validation_error = (predicted_validation - y_train[testcv])**2
        avg_validation_error += np.mean(validation_error)/4
    train_errors.append(avg_train_error)
    validation_errors.append(avg_validation_error)
